#### 自己紹介

```

また、同席している[チームメンバー名]は[役職]を担当しており、技術的な質問にお答えします。

本日は、お忙しい中お時間をいただきありがとうございます。

```

```

```

## 5. デモ後のフォローアップ計画

### 5.1 即日フォローアップ

- デモ資料の送付
- 未回答の質問への回答提供
- お礼状の送付
- 次回ミーティングの日程調整

### 5.2 短期フォローアップ（1週間以内）

- 詳細な提案書/見積りの送付
- 追加情報の提供
- 技術チームとのミーティング設定
- フィードバックの収集

- PoC/トライアルの提案
- 契約条件の詳細協議
